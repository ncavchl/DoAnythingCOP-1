## Docker 설치하기

```bash
curl -s https://get.docker.com | sudo sh  # 리눅스 배포판에 따라 자동으로 최신 도커 설치
sudo usermod -aG docker ubuntu			  # ubuntu 경우 유저권한 추가
```

### 설치 확인

```bash
$ docker version
```
